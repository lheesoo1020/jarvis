# Jarvis — Personal AI Assistant

## Project Overview
Voice-activated personal AI assistant inspired by Iron Man's Jarvis.
Built with Python, powered by Claude API.

## Core Features
- Wake word detection ("Hey Jarvis")
- Voice conversation via STT/TTS pipeline
- Personal assistant functions (calendar, weather, news)
- Long-term memory across sessions (TBD)

## Tech Stack
- **Language**: Python 3.12
- **LLM**: Claude (Api)
- **STT**: OpenAI Whisper (local)
